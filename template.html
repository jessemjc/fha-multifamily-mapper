Firm Commitment Pipeline Deals

Off by default. Adds two kinds of purple pins: recently-insured properties HUD's main file hasn't caught up to yet, and Firm Commitments that haven't closed yet — these can still fall through. The age limit only applies to the latter (not-yet-closed commitments can go stale; recently-insured ones don't need an age check) and defaults to no limit — old, likely-dead commitments are included unless you choose to exclude them. Only city-level location is available for either (no street address exists yet), shown as a coarser, distinctly-colored pin.

Data: HUD Insured Multifamily Mortgages (active) cross-referenced with HUD's Insured Active Multifamily FHA Property Addresses file by FHA project number, plus HUD's Active Multifamily Portfolio data for subsidy/restriction/tax-credit flags.

exact = geocoded to the real street address, on demand, only for properties in your current results. Tries the U.S. Census geocoder first (two different parsers), then OpenStreetMap as a last resort — different address databases catch different addresses.
approx = none of those could match the address; falling back to the ZIP code centroid.
manual = you corrected this pin by hand.
pipeline = a Firm Commitment or recently-insured deal not yet in the main HUD dataset — city-center location only (see "Firm Commitment Pipeline Deals" toggle above).

★ = the address/location you searched for. The search radius, the drawn area, and the city/state/zip/county boundary (via U.S. Census TIGERweb — may not draw if that service is briefly unavailable) all show the same way: a solid green outline.

Every pin always shows its own property number. If two properties sit right on top of each other on screen (e.g. sharing a zip centroid), they're nudged apart slightly with a thin line back to the true spot. Click any table row to jump the map straight to that property and open its details.

County names are derived from each property's ZIP code (not HUD's own county field, which has gaps), so county search is complete and consistently spelled for every property. Searching by county also draws its boundary outline, same as state/city/zip.
